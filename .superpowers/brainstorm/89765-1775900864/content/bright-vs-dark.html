Dark Maroon Base

Dark sections use #4A1010. Brand red #E72A2F as accent. More depth and contrast between sections. Buttons pop against dark backgrounds.

Key Difference

Bright Red Base
  • + Unmistakably red, bold brand presence
  • + Matches IAMAI energy
  • - Buttons need alternate styling (white or dark) to stand out on red
  • - Large red areas can feel intense
  • - Smaller text contrast is borderline WCAG
Dark Maroon Base
  • + Strong contrast — red accent pops
  • + Better text readability on dark sections
  • + Two-tone depth (maroon + bright red)
  • - Less immediately "red" at first glance
  • - More similar to current dark layout structure
